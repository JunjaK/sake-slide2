# 소주, 일본주 종류 소개

<br/>

### 일본주(日本酒：にほんしゅ 니혼슈)
<span v-mark.red="1"><code>쌀</code>, <code>누룩</code>, <code>효모</code>, <code>물</code>이 기본 재료</span>로 사용되며  
종류에 따라 <code>주정</code>(양조용 알코올), 유자 등의 과일이 들어가기도 함  
청주(清酒: せいしゅ 세이슈)로도 불림  

- 종류: 준마이, 긴죠, 혼죠조 등등
